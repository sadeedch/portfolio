# Sadeed Ahmad Portfolio

A production-ready Next.js portfolio website.

## Run locally

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.

## Deploy on Vercel

1. Push this folder to GitHub.
2. Import the repository in Vercel.
3. Click Deploy.

## Main files

- `app/page.js` contains the portfolio UI.
- `app/layout.js` contains SEO metadata.
- `app/globals.css` contains global styling.
